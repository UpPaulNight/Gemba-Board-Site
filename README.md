# Gemba Board Site

This repository contains HTML file and supporting assets for the Gemba Boards.

It contains

- The HTML files for the Gemba Board web interface.
- A simple Python HTTP server to serve the files.
- Systemd service and timer files to automate refreshing the content and
  starting the server and browser on login.
